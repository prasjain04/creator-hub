# Creator Partnership Hub

A tool for TikTok creators to manage brand partnerships.

## Setup

1. Clone this repo
2. Run `npm install`
3. Run `npm run dev` to start locally

## Deploy to Vercel

1. Push to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Add environment variable:
   - `ANTHROPIC_API_KEY` = your API key from [console.anthropic.com](https://console.anthropic.com)
4. Deploy!

## Get an Anthropic API Key

1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Sign up / Sign in
3. Go to API Keys → Create Key
4. Copy the key and add it to Vercel

## Features

- Generate brand partnership suggestions from your TikTok scripts
- AI-powered personalized outreach messages
- Kanban board to track partnership status
- Track deal amounts and deliverables
- Payment status tracking
